## 使用说明

1. **生成后处理**：用 Photoshop/GIMP 将横排精灵图按 192x192 切片
2. **Unity 导入设置**：
   - Pixels Per Unit: 64
   - Filter Mode: Point (No Filter)
   - Compression: None
   - Sprite Mode: Multiple
3. **Animator Controller**：参考现有的 Player.controller 结构，添加 Attack/Guard 状态
4. **颜色方案**：
   - 无名：黑+红+金（主色调）
   - 残剑：白+灰+墨（水墨感）
   - 飞雪：红+白+粉（丝绸感）
   - 地图：黑+红+金+绿（秦朝宫廷+自然）
